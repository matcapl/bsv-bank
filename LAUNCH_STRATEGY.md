# BSV Bank - Launch Strategy

## Phase 1: Solo Testing (Week 1-2)
- [ ] Deposit 0.1 BSV from your wallet
- [ ] Create 3 different sized deposits
- [ ] Test lock periods (7d, 30d, 90d)
- [ ] Verify interest accrues correctly
- [ ] Document all transactions

## Phase 2: Friends Circle (Week 3-4)
- [ ] Invite 5 close friends
- [ ] Each deposits $10-20 worth BSV
- [ ] Test P2P lending:
  * Friend A lends to Friend B
  * Verify collateral locks
  * Test repayment flow
- [ ] Collect feedback

## Phase 3: Early Adopters (Month 2)
- [ ] Launch on BSV community forums
- [ ] Target: 50 users
- [ ] Offer: Bonus APY for early adopters
- [ ] Build: Public dashboard showing TVL
- [ ] Create: Video tutorial

## Phase 4: Public Beta (Month 3-6)
- [ ] Security audit completion
- [ ] Marketing campaign
- [ ] Target: 500 users, $50k TVL
- [ ] Partnerships with BSV wallets

## Success Metrics

**Week 1:**
- ✓ 1 user (you)
- ✓ $100 TVL
- ✓ System stable

**Month 1:**
- Target: 10 users
- Target: $1,000 TVL
- 5+ lending transactions

**Month 3:**
- Target: 100 users
- Target: $10,000 TVL
- Self-sustaining interest pool

**Month 6:**
- Target: 500 users
- Target: $100,000 TVL
- Profitable operation
